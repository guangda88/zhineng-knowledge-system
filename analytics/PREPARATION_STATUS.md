## 📞 问题排查

### 常见问题

**Q1: 脚本执行失败，提示找不到模块**
```bash
A: 确保Python路径正确
   export PYTHONPATH=/home/ai/zhineng-knowledge-system/services
```

**Q2: 数据库连接失败**
```bash
A: 检查PostgreSQL服务状态
   docker-compose ps postgres
   docker exec tcm-postgres pg_isready -U tcmuser
```

**Q3: 后端服务未响应**
```bash
A: 检查后端容器状态
   docker-compose ps backend
   docker logs tcm-backend
```

**Q4: 数据生成很慢**
```bash
A: 减少生成的数据量
   或增加数据库连接池大小
```
