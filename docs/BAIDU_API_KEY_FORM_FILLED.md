```
应用描述: 智能知识库系统，集成千帆大模型用于文本生成、知识问答、任务处理和推理增强
```

```
应用描述: 灵知智能知识库系统生产环境，使用千帆ERNIE-Bot模型提供AI能力

curl ifconfig.me
```

### 方案3: 开发测试

```

应用描述: 灵知系统开发测试，用于验证千帆API集成和功能测试
```

## 其他可选字段

### 如果有"每日调用量上限"：

```
每日调用量上限: 1000000
(留空或不填，使用免费额度即可)
```

### 如果有"回调URL"：

```
留空（后端应用不需要）
```

### 如果有"应用首页"：

```
留空（后端应用不需要）
```

## 创建后立即保存

创建成功后会显示：

```
┌──────────────────────────────────────────────┐
│ API Key创建成功                              │
├──────────────────────────────────────────────┤
│                                              │
│ API Key:                                     │
│ ┌────────────────────────────────────────┐  │
│ │ ak-xxxxxxxxxxxxxxxxxxxxxxxx            │  │
│ └────────────────────────────────────────┘  │
│ [复制]                                      │
│                                              │
│ Secret Key: (只显示一次，务必立即复制保存)   │
│ ┌────────────────────────────────────────┐  │
│ │ sk-xxxxxxxxxxxxxxxxxxxxxxxx            │  │
│ └────────────────────────────────────────┘  │
│ [复制]                                      │
│                                              │
│ ⚠️ Secret Key离开此页面后将无法再次查看！    │
│                                              │
│            [我已保存，关闭]                  │
└──────────────────────────────────────────────┘
```

**重要操作：**
1. 点击"复制"按钮复制 API Key
2. 点击"复制"按钮复制 Secret Key
3. 保存到密码管理器或安全文件
4. 点击"我已保存，关闭"

---

## 复制到系统

保存成功后，运行：

```bash
# 替换.env中的占位符
nano .env

# 找到这一行：
QWEN_API_KEY=your_qwen_api_key_here

# 替换为：
QWEN_API_KEY=ak-你刚才复制的API_Key

# 保存并退出（Ctrl+X, Y, Enter）
```

**只需API Key即可，Secret Key暂时不需要！**
