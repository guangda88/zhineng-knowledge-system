## 系统简介

智能知识系统是一个基于人工智能的传统文化知识问答平台，专注于：

- **气功**: 各种气功功法、练习方法、养生知识
- **中医**: 中医理论、诊断方法、中药知识
- **儒家**: 儒家思想、经典著作、修身理念

### 核心功能

| 功能 | 说明 |
|------|------|
| 知识搜索 | 关键词搜索和语义检索 |
| 文档浏览 | 分类浏览知识文档 |
| 智能问答 | 自然语言问答交互 |
| 推理分析 | 多步骤推理和知识关联 |
| 知识图谱 | 可视化知识关系网络 |

### 系统特点

- **准确可靠**: 基于精心整理的权威资料
- **智能关联**: 自动发现知识间的关联关系
- **多领域覆盖**: 气功、中医、儒家三大领域
- **交互友好**: 简洁直观的用户界面

## 功能使用

### 知识搜索

#### 基础搜索

1. 点击"搜索"标签
2. 在搜索框中输入关键词
3. 选择分类（可选）
4. 点击"搜索"按钮

**搜索技巧**:

| 技巧 | 说明 | 示例 |
|------|------|------|
| 精确匹配 | 使用完整词组 | "八段锦第一式" |
| 多关键词 | 空格分隔 | "气功 呼吸法" |
| 分类筛选 | 选择特定领域 | 选择"气功"分类 |

#### 高级搜索

系统支持混合检索，结合关键词和语义理解：

- **关键词匹配**: 精确查找文档中的词句
- **语义检索**: 理解问题含义，找到相关内容
- **自动融合**: 两种方式结合，返回最相关结果

**示例问题**:

- "八段锦的功效是什么？"
- "中医如何调理脾胃？"
- "儒家修身的核心是什么？"

### 文档浏览

#### 浏览文档列表

1. 点击"文档"标签
2. 使用分类筛选器选择领域
3. 点击文档标题查看详情

#### 文档详情

文档详情页面包含：

- **标题**: 文档名称
- **分类**: 所属领域
- **标签**: 内容标签
- **内容**: 完整文档内容
- **创建时间**: 添加到知识库的时间

### 智能问答

#### 发起提问

1. 点击"问答"标签
2. 在输入框中输入问题
3. 点击"发送"按钮

**提问技巧**:

| 类型 | 示例 | 说明 |
|------|------|------|
| 事实类 | "八段锦有几式？" | 询问具体信息 |
| 方法类 | "如何练习站桩？" | 询问操作方法 |
| 比较类 | "中医和西医的区别？" | 比较不同概念 |
| 原理类 | "气功为什么会有效？" | 询问原因原理 |

#### 多轮对话

系统支持会话连续性，可以基于之前的回答继续追问：

**示例对话**:

```
用户: 八段锦第一式是什么？
系统: 八段锦第一式是双手托天理三焦，动作要领是...

用户: 这个动作有什么功效？
系统: [理解"这个动作"指第一式] 双手托天理三焦的主要功效是...
```

### 推理分析

推理功能适用于复杂问题，系统会进行多步骤分析。

#### 推理模式

| 模式 | 适用场景 | 特点 |
|------|---------|------|
| 自动选择 | 系统判断 | 推荐使用 |
| 链式推理 (CoT) | 逻辑推理 | 逐步推理过程 |
| ReAct 模式 | 多步骤问题 | 动态调整思路 |
| 图谱推理 | 关系分析 | 利用知识图谱 |

#### 使用方法

1. 点击"推理"标签
2. 选择推理模式（默认自动）
3. 选择领域（可选）
4. 输入复杂问题
5. 点击"开始推理"

**示例问题**:

```
"八段锦和太极拳在养生效果上有什么异同？"
"中医的阴阳理论如何应用于气功练习？"
"儒家的修身思想对中医养生有什么影响？"
```

#### 推理结果

推理结果包含：

- **推理步骤**: 展示思考过程
- **最终答案**: 综合分析后的结论
- **参考来源**: 依据的知识文档

### 知识图谱

知识图谱以可视化方式展示知识之间的关联关系。

#### 查看图谱

1. 点击"推理"标签
2. 滚动到知识图谱区域
3. 点击"构建图谱"按钮（首次使用）

#### 图谱交互

- **节点**: 代表概念、实体
- **连线**: 代表关系
- **拖拽**: 移动节点位置
- **点击**: 查看实体详情

#### 图谱查询

使用图谱查询功能可以：

- 查找两个概念之间的关系路径
- 发现间接关联
- 理解知识网络结构

**示例查询**:

```
起始实体: 八段锦
目标实体: 气功
最大深度: 3

结果: 八段锦 -> 属于 -> 气功功法 -> 属于 -> 气功
```

## 使用场景

### 场景一: 气功练习

**需求**: 了解气功功法的练习方法

**操作流程**:

1. 进入"搜索"标签
2. 搜索"八段锦"
3. 浏览搜索结果
4. 查看具体文档内容
5. 如有疑问，进入"问答"继续提问

**常见问题**:

- "八段锦每式练习多长时间？"
- "气功初学者应该从哪个功法开始？"
- "练习气功有哪些注意事项？"

**需求**: 学习中医养生理论

**操作流程**:

1. 选择"中医"分类
2. 浏览相关文档
3. 使用"问答"功能深入理解
4. 使用"推理"功能分析不同概念的关系

**常见问题**:

- "中医如何判断体质类型？"
- "不同体质如何调理？"
- "中医养生的基本原则是什么？"

### 场景三: 儒家学习

**需求**: 理解儒家思想体系

**操作流程**:

1. 浏览"儒家"分类文档
2. 使用"图谱推理"理解概念关系
3. 通过"问答"澄清疑问

**常见问题**:

- "什么是格物致知？"
- "儒家修身的核心是什么？"
- "儒家思想如何应用于现代生活？"

### 场景四: 跨领域研究

**需求**: 研究气功、中医、儒家之间的关联

**操作流程**:

1. 使用"推理"功能
2. 选择"图谱推理"模式
3. 提出跨领域问题
4. 查看知识图谱发现关联

**示例问题**:

- "儒家的中庸思想如何体现在中医理论中？"
- "气功练习与中医的气血理论有什么关系？"

## 常见问题

### 使用问题

#### Q: 搜索不到相关内容怎么办？

A: 尝试以下方法：

1. 使用不同的关键词
2. 切换到其他分类
3. 使用更通用的词汇
4. 使用"问答"功能直接提问

#### Q: 回答不准确怎么办？

A: 可以尝试：

1. 问题表达得更清晰
2. 使用"推理"功能获得更深入分析
3. 分解复杂问题，逐个提问
4. 指定相关领域

#### Q: 知识图谱显示不完整？

A: 这是正常现象：

1. 图谱只显示已建立关联的内容
2. 可以点击"构建图谱"更新
3. 随着知识库增长会越来越丰富

#### Q: 如何获得更好的推理结果？

A: 优化提问方式：

1. 描述清楚问题的背景
2. 明确要比较的对象
3. 选择合适的推理模式
4. 指定相关领域

### 功能问题

#### Q: 支持语音输入吗？

A: 当前版本暂不支持，计划在后续版本添加。

#### Q: 可以保存对话历史吗？

A: 系统会临时保存会话记录，刷新页面后会重置。

#### Q: 支持哪些语言？

A: 目前主要支持中文，英文支持正在开发中。

#### Q: 可以导出搜索结果吗？

A: 可以复制内容到剪贴板，导出功能正在开发。

### 内容问题

#### Q: 知识内容的权威性如何？

A: 系统内容来源于：

- 正版教材和专著
- 权威机构的公开资料
- 专家审核的文献

#### Q: 内容会持续更新吗？

A: 是的，知识库会定期更新和扩充。

#### Q: 如何判断内容是否适用于我的情况？

A: 建议：

1. 参考多个来源
2. 咨询专业人士
3. 结合自身情况判断

## 最佳实践

### 提问技巧

1. **明确具体**: 避免模糊的问题
   - 好: "八段锦第一式的动作要领是什么？"
   - 差: "八段锦怎么样？"

2. **提供背景**: 帮助系统理解上下文
   - 好: "气功初学者适合练习什么功法？"
   - 差: "练什么？"

3. **一次一问**: 避免在一个问题中包含多个疑问
   - 好: "八段锦有哪些功效？" 然后 "太极拳有哪些功效？"
   - 差: "八段锦和太极拳分别有什么功效和区别？"

4. **使用标准术语**: 使用领域内的标准表述
   - 好: "气功的调息方法是什么？"
   - 差: "气功怎么呼吸？"

### 学习建议

1. **循序渐进**: 从基础概念开始，逐步深入

2. **多方验证**: 对于重要的知识，多方查证

3. **理论结合实践**: 学习理论后注意实践应用

4. **保持开放**: 传统文化知识丰富，保持学习心态
