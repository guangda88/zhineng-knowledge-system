**问题描述**: tcm-xinference 容器持续重启，无法正常运行

**原因分析**: 
- 容器启动命令错误：`/bin/bash`
- `/bin/bash` 是交互式 shell，没有输入时立即退出
- 导致容器退出，触发重启策略

**修复方案**:
1. 停止并删除错误的容器
2. 使用正确的启动命令重新创建容器
3. 正确的启动命令: `xinference-local --host 0.0.0.0 --port 9997`

**修复结果**:
