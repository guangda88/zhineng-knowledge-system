# 人机交互标注系统说明

**日期**: 2026年3月31日
**目的**: 澄清标注系统的价值定位

---

## 系统目的

**提升图片和音视频的识别精度**

---

## 为什么识别精度很重要

### 1. 确保知识准确性

```
古籍图片识别
❌ 识别不准确：
"虚凌顶劲" → "虚心顶劲"
用户学到错误动作

✅ 识别准确（通过标注纠正）：
"虚凌顶劲" → "虚凌顶劲"
用户学到正确动作
```

### 2. 避免错误练习

```
音频转写
❌ 转写不准确：
"意念集中丹田" → "意念集中胆田"
用户找不到正确位置

✅ 转写准确（通过标注纠正）：
"意念集中丹田" → "意念集中丹田"
用户找到正确位置
```

### 3. 确保练习有效

```
视频理解
❌ 理解不准确：
"身体保持中正" → "身体前倾"
用户动作错误

✅ 理解准确（通过标注纠正）：
"身体保持中正" → "身体保持中正"
用户动作正确
```

## 用户关心识别精度吗？

### 用户表面关心的问题

- "这个动作怎么做才正确？"
- "练习时要注意什么？"
- "如何知道自己做对了？"

### 用户隐含关心的基础

- ✅ "我学到的是正确的吗？" ← **依赖识别准确**
- ✅ "这个指导准确吗？" ← **依赖识别准确**
- ✅ "我可以相信这个内容吗？" ← **依赖识别准确**

### 结论

**用户关心"是否有效"，而"识别准确"是"有效"的前提**
