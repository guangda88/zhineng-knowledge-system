# 📚 智能知识系统 - 前端使用指南

**访问地址**: http://100.66.1.8:8008

---

## ✨ 功能概览

### 🔍 知识搜索
- **功能**: 搜索知识库中的相关文档
- **使用**: 输入关键词，选择分类筛选
- **返回**: 相关文档列表

### 📄 文档管理
- **功能**: 浏览和管理所有文档
- **使用**: 切换分类查看文档列表

### 📚 **书籍搜索** (新功能)
- **功能**: 搜索古籍书籍的元数据和章节内容
- **特性**:
  - 📖 **元数据搜索**: 按书名、作者搜索
  - 📄 **全文搜索**: 在章节内容中搜索
  - 🏷️ **分类筛选**: 气功、中医、儒家
  - 📜 **朝代筛选**: 从先秦到清朝
  - 📖 **书籍详情**: 查看完整书籍信息和章节
  - 🔗 **相关推荐**: 基于相似度的书籍推荐

### 💬 智能问答
- **功能**: 提问并获得基于RAG的答案
- **使用**: 输入问题，查看对话历史

### 🧠 推理系统
- **功能**: 复杂问题的多步推理
- **模式**: 自动选择、CoT、ReAct、图谱推理

## 📚 书籍搜索功能详解

### 搜索类型

#### 1. 📖 元数据搜索
搜索范围：书名、作者、描述

**示例查询**:
- `周易` → 查找《周易注疏》
- `孔子` → 查找《论语》
- `老子` → 查找《道德经》

**筛选选项**:
- 分类: 气功、中医、儒家
- 朝代: 先秦、春秋、战国、秦汉、魏晋、唐、宋、元、明、清

#### 2. 📄 全文搜索
搜索范围：书籍章节内容

**适用场景**:
- 查找特定概念在书籍中的位置
- 发现包含特定主题的章节

---

### 使用步骤

1. **访问页面**
   - 打开 http://100.66.1.8:8008
   - 点击"📚 书籍"标签

2. **选择搜索类型**
   - 📖 书籍搜索（默认）：搜索书名、作者
   - 📄 全文搜索：搜索章节内容

3. **输入搜索词**
   - 在搜索框中输入关键词
   - 如：`周易`、`论语`、`气功`

4. **筛选（可选）**
   - 选择分类：气功、中医、儒家
   - 选择朝代：如"春秋"、"魏晋"

5. **查看结果**
   - 显示书籍卡片，包含：
     - 书名和作者
     - 分类和朝代标签
     - 简短描述
     - 阅读统计

6. **查看详情**
   - 点击书籍卡片查看完整信息
   - 包含：
     - 详细描述
     - 章节列表
     - 阅读次数

7. **阅读章节**
   - 点击章节名称查看完整内容
   - 模态框显示章节文本

8. **相关推荐**
   - 在书籍详情中点击"相关推荐"
   - 基于向量相似度推荐相关书籍
   - 显示相似度百分比

## 🎯 搜索技巧

### 精确搜索
- 输入完整书名：`周易注疏`
- 输入作者名：`老子`、`孔子`

### 模糊搜索
- 输入部分关键词：`周` → 匹配"周易注疏"
- 支持模糊匹配：`孔孟` → 可能匹配包含"孔"或"孟"的结果

### 分类筛选
- 单独查看某类书籍：选择"儒家"
- 查看特定朝代：选择"魏晋"

### 组合搜索
- 搜索 + 分类：`周易` + "儒家"
- 搜索 + 朝代：`孔子` + "春秋"

## 🔧 API接口

### 基础URL
```
http://100.66.1.8:8008/api/v2/library
```

### 端点列表

| 端点 | 方法 | 功能 | 参数 |
|------|------|------|------|
| `/search` | GET | 元数据搜索 | `q`, `category`, `dynasty`, `author`, `page`, `size` |
| `/search/content` | GET | 全文搜索 | `q`, `category`, `page`, `size` |
| `/{book_id}` | GET | 书籍详情 | 书籍ID |
| `/{book_id}/related` | GET | 相关推荐 | `book_id`, `top_k`, `threshold` |
| `/{book_id}/chapters/{chapter_id}` | GET | 章节内容 | 书籍ID, 章节ID |

### 请求示例

```bash
# 搜索书籍
curl "http://100.66.1.8:8008/api/v2/library/search?q=周易"

# 按分类筛选
curl "http://100.66.1.8:8008/api/v2/library/search?category=儒家"

# 获取书籍详情
curl "http://100.66.1.8:8008/api/v2/library/2"

# 相关推荐
curl "http://100.66.1.8:8008/api/v2/library/2/related"
```

## 🐛 故障排除

### 问题：搜索无结果
**解决方案**:
- 尝试更简单的关键词
- 检查分类/朝代筛选是否过于严格
- 尝试空搜索查看所有书籍

### 问题：页面不显示
**解决方案**:
- 刷新浏览器页面 (F5)
- 清除浏览器缓存
- 检查控制台错误信息

### 问题：点击无反应
**解决方案**:
- 确保JavaScript已启用
- 检查浏览器控制台是否有错误
- 刷新页面重试
