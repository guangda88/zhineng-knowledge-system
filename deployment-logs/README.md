# 部署日志目录

本目录用于存储所有部署活动的日志记录。

## 文件命名规范

```
deploy_YYYYMMDD_HHMMSS.md
```

例如: `deploy_20240325_103000.md`

## 日志内容

每次部署应创建一个新的日志文件，使用模板：
`../templates/deploy_log_template.md`

## 日志索引

| 日期 | 文件 | 类型 | 执行人 | 结果 |
|------|------|------|--------|------|
| 2024-03-25 | deploy_20240325_103000.md | 标准部署 | [姓名] | ✅ 成功 |

## 查看日志

\`\`\`bash
# 列出所有部署日志
ls -lt deploy_*.md

# 查看最近一次部署
cat \$(ls -t deploy_*.md | head -1)

# 搜索失败的部署
grep -l "失败" deploy_*.md
\`\`\`
