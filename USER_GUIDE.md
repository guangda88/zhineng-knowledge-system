## 快速开始

### 系统要求

| 组件 | 要求 |
|------|------|
| Docker | 24.0+ |
| Docker Compose | 2.20+ |
| 内存 | 最低 4GB |
| 磁盘 | 最低 10GB |

### 启动系统

```bash
# 1. 克隆项目
git clone https://github.com/guangda88/zhineng-knowledge-system.git
cd zhineng-knowledge-system

# 2. 配置环境变量（可选）
cp .env.example .env
# 编辑 .env 文件配置API密钥等

# 3. 启动所有服务
docker-compose up -d

# 4. 查看服务状态
docker-compose ps

# 5. 查看日志（如有问题）
docker-compose logs -f backend
```

### 各功能模块详解

#### 1. 搜索 (🔍)

**用途**: 全库语义搜索，支持气功、中医、儒学三大领域

**特点**:
- 向量语义检索
- BM25关键词检索
- 混合双路召回

#### 2. 文档 (📄)

**用途**: 浏览系统中的所有知识文档

**支持操作**:
- 按分类筛选
- 分页浏览
- 查看文档详情

#### 3. 书籍 (📚)

**用途**: 古籍书名、作者、朝代检索

**搜索模式**:
- 书籍搜索：元数据检索
- 全文搜索：内容检索

#### 4. 国学 (📜)

**用途**: 检索国学经典原文

**支持典籍**: 论语、道德经、史记、易经、左传、庄子、孟子、荀子

#### 5. 书目 (📖)

**用途**: 检索302万条书目数据

**筛选条件**: 领域、文件格式

#### 6. 问答 (💬)

**用途**: AI智能问答对话

**特点**: 理论+科学+实践完整回答

#### 7. 推理 (🧠)

**用途**: 深度推理分析

**模式**: CoT、ReAct、图谱推理

## 常见问题

### Q1: 搜索没有结果怎么办？

```
问题排查流程:

1. 检查关键词
   └─> 是否太专业？尝试更通用的词

2. 检查分类筛选
   └─> 尝试切换或取消分类

3. 尝试其他搜索方式
   └─> 使用问答功能直接提问

4. 尝试全文搜索
   └─> 书籍搜索切换到全文模式
```

### Q2: AI回答不准确？

```
优化方法:

1. 问题更具体
   ❌ "气功"
   ✅ "气功的三调是什么？"

2. 提供上下文
   ❌ "怎么练？"
   ✅ "初学者如何练习站桩？"

3. 尝试推理模式
   └─> 使用CoT获得更深入的分析

4. 切换分类
   └─> 选择更相关的分类
```

### Q3: 如何找到实践指导？

```
搜索策略:

1. 使用实践关键词
   - "练习"、"方法"、"如何"
   - "入门"、"基础"、"教程"

2. 问答直接询问
   - "如何练习X？"
   - "X的步骤是什么？"

3. 查看文档实践章节
   └─> 文档列表中有标注
```

### Q4: 系统响应慢？

```
排查步骤:

1. 检查服务状态
   ```bash
   docker-compose ps
   docker-compose logs backend
   ```

2. 减少返回数量
   └─> 将limit从100降到20

3. 使用更精确的关键词
   └─> 提高搜索精度

4. 非高峰时段使用
   └─> 避开高并发时段
```

### Q5: 如何验证搜索结果？

```
验证方法:

1. 查看来源信息
   └─> 结果中包含来源书籍

2. 对比多个来源
   └─> 使用不同搜索方式交叉验证

3. 使用图谱推理
   └─> 查看知识关联关系

4. 查看原始书籍
   └─> 书籍搜索后查看原文
```

### 技巧1: 组合搜索

```
多关键词组合:
"太极 气功" → 搜索同时包含两个词的内容
"松静 自然" → 搜索短语

分类+关键词:

精确匹配:
"三调" → 双引号精确匹配
```

### 技巧2: 递进式查询

```
第一层: 广泛搜索
搜索: "气功"
↓
第二层: 缩小范围
搜索: "气功 入门"
↓
第三层: 精确定位
搜索: "气功 入门 三调"
↓
第四层: 深入理解
问答: "什么是气功的三调？"
```

### 技巧3: 对比研究

```
1. 搜索两个概念
   搜索: "太极拳 太极气功"

2. 分别了解
   问答: "什么是太极拳？"
   问答: "什么是太极气功？"

3. 对比分析
   推理: "太极拳和太极气功的区别？"
